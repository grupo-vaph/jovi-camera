Comp. Thinking with Python - Aula 05

  Estrutura de Repetição: while
  
  O comando while repete um bloco de código enquanto a condição for True.
  A cada iteração, a condição é reavaliada.

  Exemplo:  

  contador = 0
  while contador < 5:
      print(contador)
      contador += 1

  Saída: 0, 1, 2, 3, 4
