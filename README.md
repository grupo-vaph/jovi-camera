# Jovi Camera

Próxima geração da experiência de câmera nos celulares Jovi.
Projeto desenvolvido no Challenge 2026 da FIAP em parceria com a Jovi (vivo Mobile).

## Como executar

Clone o repositório e abra o `index.html` em qualquer navegador moderno:

```bash
git clone https://github.com/grupo-vaph/jovi-camera.git
cd jovi-camera
open index.html
```

Não há dependências externas. Projeto desenvolvido em HTML, CSS e JavaScript puro.
